# iTerm Italic Dark

A VS Code dark theme inspired by iTerm color contrast, with italic keywords and vivid syntax accents.

## Highlights

- Deep black editor and terminal background for focus.
- Improved panel and sidebar contrast so sections are easier to scan.
- Italic keywords and comments for a classic terminal-aesthetic coding style.
- Expanded workbench coverage for tabs, lists, inputs, status bar, and diff colors.
- Semantic token support enabled for modern language servers.

## Install (Development)

1. Open this folder in VS Code.
2. Press `F5` to launch an Extension Development Host.
3. In the new window, open Command Palette and run `Preferences: Color Theme`.
4. Select `iTerm Italic Dark`.

## Notes

- If italics do not appear, set a font that supports italics in editor settings.
- Theme file: `themes/iTerm Italic Dark-color-theme.json`.
